Introduction

The Heliotis heliCam C3 is an optical imaging system, which offers 3D image data in real time.

This document outlines the software API for Heliotis cameras. More general information on the setup and use of the camera may be found in the corresponding user manual. Please contact Heliotis for details.

Installed files and location

The described directories are for Windows7 (64bit) default installation.
Other platform can have a slightly different folders

C:/Program Files/Heliotis/heliCam
The directory Cpp contains C++ classes, examples and documentations
The directory Halcon contains the Halcon interface, examples and documentations
The directory LabView contains LabView library, examples and documentations
The directory libHeLIC contains the libHeLIC API header and library files
The directory Python contains the Python wrapper, examples and documentations
The directory sys contains the USB drivers and other system files

C:/ProgramData/Heliotis
The directory devDesc contains description files of the system and registers.
SysDesc.xml is the system description file. It is linked to the Reg_c3cam.xml files, which include the description of all registers.
The directory devDesc/fpga contains the FPGA firmware, that is downloaded if needed.
The directory devDesc/i2c contains FX2 firmware, that is downloaded if needed.

Sample Code

Heliotis has various examples in C++, Python and LabVIEW
You find the examples and a short documentation about it in the following folders:
C++: C:/Program Files/Heliotis/heliCam/Cpp
Python: C:/Program Files/Heliotis/heliCam/Python
LabVIEW: C:/Program Files/Heliotis/heliCam/LabView
