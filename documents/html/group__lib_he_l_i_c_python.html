Python Interface

Python is supported on various platform.

The Heliotis libHeLIC Python module makes all function of the libHeLIC API available to Python.
The used Python module is: libHeLIC.py, located at C:/Program Files/Heliotis/heliCam/Python/wrapper
Further there are some example and test code located at C:/Program Files/Heliotis/heliCam/Python/example

Mainly all HE_functions from libHeLIC are available with that wrapper in python. They can be called from a LibHeLIC instance without the HE_ extention.

Here a short code snippet to access the library:

import os, sys
prgPath=os.environ["PROGRAMFILES"]
sys.path.insert(0,prgPath+r'\Heliotis\heliCam\Python\wrapper')
from libHeLIC import *
heSys=libHeLIC()
heSys.Open(0,sys='c3cam_sl70')
heSys.Acquire()
img=heSys.GetCamData(0,0,0)
heSys.Close()
