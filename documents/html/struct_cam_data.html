Detailed Description

CamData is a data buffer descriptor where date is acquired from the camera or processed to
CamData buffers can be handled with HE_AllocCamData(), HE_GetCamData(), HE_SetCamData(), HE_FreeCamData() etc.
A CamData buffer is a pointer that can change, if the buffer size changes. This can be done if the number of frames or if the mode is modified. Therefore it is recommended to use HE_GetCamData() with the index to get the current pointer.
The buffer with index 0 is always the raw data buffer, that contains the data as it comes from the camera.
If the CamData has the property DP_DATA_EXTERN, the buffer is managed externally. For such buffers reallocation can not be achieved automatically and the user is responsible, that the memory is big enough.
