Detailed Description

The Heliotis camera can be accessed from LabView.

For that purpose there is a LabView library for the camera.

Examples are located at C:/Program Files/Heliotis/heliCam/LabView/example
A LabView documentation are located at C:/Program Files/Heliotis/heliCam/LabView/documents

All libHeLIC functions can also be accessed directly trough the LabView-dll nodes.
For LabView there exists some special functions to manage error and memory handling as described below.
These specia LabView functions starts with a LV_ prefix.

LabView acquisition function. Acquires data from the camera and stores them in appropriate LabView arrays.
dataArrHdl . encoderArrHdl is a one dimensional array of 2kB Heaser informations that contains mainly the encoder values.

Parameters
heHdlHandle to the camera
fmtin: result data format see also: CamDataFmt
propin: result data property see also: CamDataProperty
dataArrHdlin/out: resulting data array handle. This array handle must have the dimensions and datatype as requested by the fmt and prop settings
encoderArrHdlin/out: resulting encoder array handle
voidMetaout: meta data of output: number of dimensions, see also examples
