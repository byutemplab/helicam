(major version).(features).(bug fixes).(build number)

Release Notes version: 1.7.5.x (19.09.2018)
  • Update Python wrapper (add support for Python 3.5 64bit)
  • Update Linux udev rules
Release Notes version: 1.7.4.x (29.06.2018)
  • Add helper library to SDK for surface extraction of camMode 5 raw data
  • Update HALCON interfaces
Release Notes version: 1.7.3.x (05.07.2017)
  • Update cpp example and library for Xenax stage
  • Update HALCON interface
  • Update Python wrapper to read camera serialnumber
Release Notes version: 1.7.2.x (24.01.2017)
  • Update C++ class for xenax controller
  • Add C++ Library for Xenax controller (static library)
  • Update HALCON interface. Add support for HALCON 13 and Bugfixes.
  • Update Python wrapper for callback functions
  • Update API and add HE_CleanUSB() function.
Release Notes version: 1.7.1.x (22.06.2016)
  • Update Intensity mode. Remove fix offset for exposure time configuration...
  • Update Python wrapper for Python 3 compatibility.
  • Add LabVIEW example for Volumen Mode.
  • Linux only: Use environment variable "HELIOTIS_CONFIGPATH" to define a custom configuration path for all configfiles (SysDesc.xml, Reg_c3cam.xml, fpga/3.rbf, i2c/*.iic) The custom path must include all this configuration files in the subfolder "devDesc". A copy from this required files must be creadted independently. If the environment variable "HELIOTIS_CONFIGPATH" not set, the default path (/usr/share/libhelic) is used!
Release Notes version: 1.7.0.x (06.05.2016)

  • Update dll functions for LabVIEW. Add a function for meta informations of camera parameters.
  • Update python script to plot measurements from Cpp example

Release Notes version: 1.4.0.x (30.03.2015)
  • Implement HE_GetSerials(char* str) function to count connected (unused) heliCams and return there serial number. Open a selected/defined heliCam with: HE_Open(&heHdl, 0, "sys=c3cam_sl70 serial=...")
